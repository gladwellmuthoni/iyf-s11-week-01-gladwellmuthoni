## Website 2: https://developer.mozilla.org (MDN Web Docs)

### Navigation Menu Tag
The main navigation menu is typically wrapped in a `<nav>` element.

### Search Bar Structure
The search bar is typically structured using a `<form>` element containing an `<input type="search">` or `<input type="text">` along with a `<button>` element for submission.

### Hover Effects
When hovering over links, the styles often change, for example:
- A change in `text-decoration` (e.g., adding an underline).
- A change in `color` or `background-color`.
- Addition of a transition effect defined in the CSS.
